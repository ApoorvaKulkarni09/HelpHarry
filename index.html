what is a console screen ?

Console is a screen where you display your output, and it is used for testing purposes. Before printing anything on the HTML page it is advisable to print on the console screen.

how to open and print text/no. on the console screen ?

To open console screen :
first run your project on the internet browser and and double click the mouse and a list of options appear on the screen , choose inspect , the console screen will be visible
OR
run your project on the internet browser and press ctrl + shift + J , the console screen will be visible
How to print on console screen :
To print any text or no. on console screen , go in your html file under script section and write console.log() and write what you want to be displayed inside the round brackets with quotation
